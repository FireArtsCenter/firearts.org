# firearts.org

> These steps are rough as sandpaper so expect many changes to come soon.

A site using Jekyll to build to GitHub Pages.

## Installation

1. Clone repo locally.
1. Install dependencies:
    - `bundle install`
    - `yarn`
1. Run `yarn develop` to develop locally.
